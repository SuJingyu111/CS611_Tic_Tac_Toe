All the source files are in the TTTGame package. The main method is located in TicTacToe.java. Get to the root directory
of TTTGame, and run the following commands to compile and execute the program:
javac TTTGame/*.java
java TTTGame.TicTacToe

TicTacToe: An overall architecture of the game. The main method is located here.
Board: The board on which the game is played.
Player: Different players of the game.
